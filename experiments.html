E = m c 2

Despite being in the HTML specification for a long time, developers could not make use of it until recently because browsers did not support it until their recent releases. Without browser support, there is no point for a developer to add MathML elements since the browser would not be able to render it. With increasing number of browsers supporting MathML recently, developers can now make use of it.
